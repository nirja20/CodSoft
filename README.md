CODSOFT Internship - Task 1: Landing Page
This is the Task 1 - Landing Page project submitted for the CODSOFT Web Development Internship.

📌 Project Description
A Landing Page is an ideal web development project for beginners. This project was built using only HTML and CSS, and focuses on mastering the foundational layout and design skills in web development.

🔍 What I Learned
Creating and organizing HTML structure with semantic tags
Using CSS for layout, spacing, and styling
Creating sections like header, hero banner, features, about, and course blocks
Applying hover effects and box shadows for interactivity
Designing a responsive and visually appealing layout
🖥️ Technologies Used
HTML5
CSS3
Google Fonts (Roboto)
🧩 Features Implemented
Top header with contact info and login button
Logo and navigation menu
Hero section with call-to-action
Feature highlights inside a styled box
About section describing the platform
Popular course offerings displayed with icons/images
Footer with copyright
